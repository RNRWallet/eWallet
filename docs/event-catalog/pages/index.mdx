## 📖 What is eHailing?

eHailing is a modern ride-sharing platform designed for public. Built using cutting-edge technologies and architectural patterns, it demonstrates best practices in:

- **Microservices Architecture**: Loosely coupled services with clear boundaries
- **Event-Driven Design**: Asynchronous communication using domain events
- **Cloud-Native Principles**: Containerized, scalable, and resilient
- **Kotlin and Quarkus Integration**: Simplified cloud-native development experience
